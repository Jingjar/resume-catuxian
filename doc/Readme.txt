*作品集計畫-

作品集除了放上課教的對發票系統、打地鼠、文字遊戲、時鐘以外的
作品集，我自己還正在做一個記帳系統。

*視覺計畫-

畫面的主要配色都是Bootstrap裡面的配色為主，希望給人一種穩重
專業的感覺。

*資料庫規劃-

資料庫包含了大頭照、希望職缺、關於我、學歷、經歷、技能、作品集
，而我在經歷跟經歷的細項做了關聯，訪客留言也有設計一個資料表存
取。

*功能規劃-

前台點擊連結的時候，畫面會滑至該區域，而後台可以對大頭照、希望
職缺、關於我、學歷、經歷、技能、作品集等內容進行編輯，作品集有
分頁功能，經歷的細項可以新增、編輯、刪除，而訪客留言也有一個頁
面，可以記錄是否已回復訪客的訊息。
